Как открыть сайт

✅ Способ 1 — HTML (самый простой)

Вы уже в HTML-версии. Откройте файл двойным щелчком:

  1. Откройте папку:
    C:\Users\I_m_already18\snt-klin\static-html
  2. Дважды щёлкните по файлу index.html
  3. Сайт откроется в браузере (Chrome, Edge, Firefox)

Для входа: petrov@example.ru / resident123

Способ 2 — Полная версия (с базой данных)

Нужен установленный Node.js.

  1. Нажмите Win + R, введите cmd, Enter
  2. В чёрном окне введите по очереди:
cd C:\Users\I_m_already18\snt-klin
npm install
npm run db:seed
npm run dev

Откройте в браузере: http://localhost:3000

Окно cmd не закрывайте — пока оно открыто, сайт работает.
